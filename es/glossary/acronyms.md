# Siglas

| Sigla   | Término completo                    | Notas/Región            |
|---------|-------------------------------------|--------------------------|
| ADA     | Americans with Disabilities Act     | Estados Unidos           |
| IEP     | Individualized Education Program    | Educación K-12 de EE. UU.|
| WCAG    | Web Content Accessibility Guidelines| Norma global             |
| SSI     | Supplemental Security Income        | Estados Unidos           |
| NDIS    | National Disability Insurance Scheme| Australia                |
